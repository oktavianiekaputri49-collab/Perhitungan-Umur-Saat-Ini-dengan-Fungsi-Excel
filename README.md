📊 Perhitungan Umur Saat Ini dengan Fungsi Excel
Deskripsi
Proyek ini berfokus pada pengolahan data tanggal menggunakan Microsoft Excel untuk menghitung umur berdasarkan tanggal lahir secara otomatis dan real-time.

Dataset
Data terdiri dari:
- Nama
- Tanggal lahir
- Hasil perhitungan umur dengan beberapa metode

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Fungsi:
  - TODAY (tanggal saat ini)
  - YEAR (mengambil tahun)
  - DATEDIF (selisih umur)
  - YEARFRAC (perhitungan usia berbasis pecahan tahun)

Hasil Analisis
- Umur dihitung secara otomatis berdasarkan tanggal saat ini
- Terdapat perbedaan hasil antar metode
- Metode tertentu lebih akurat dalam menghitung usia

Tujuan Proyek
- Memahami berbagai cara menghitung umur di Excel
- Membandingkan hasil dari beberapa fungsi
- Melatih pengolahan data berbasis waktu

Insight
Perhitungan umur dapat menghasilkan nilai berbeda tergantung metode yang digunakan, sehingga penting memilih fungsi yang sesuai agar hasil lebih akurat.

Penutup
Proyek ini merupakan latihan analisis data berbasis waktu dengan pendekatan perbandingan metode di Excel.
